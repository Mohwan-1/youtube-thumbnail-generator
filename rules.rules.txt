# RULES – 썸네일 자동 생성기

## MVP 원칙
- 복잡한 기능 최소화, 빠른 배포와 수익화가 목표
- 썸네일 생성은 텍스트+배경 수준 (고급 편집 기능 X)

## 광고 규칙
- 자동광고: 좌/우 사이드바
- 수동광고: 상단/중간/하단
- 광고는 사용자 경험을 방해하지 않는 선에서 배치

## 디자인 규칙
- 기본 다크모드
- 현대적, 심플 UI (반응형 지원)
- 로고와 파비콘 필수 적용

## API 규칙
- Google Gemini API 사용
- 사용자가 직접 API KEY 입력
- KEY 발급 방법 안내 버튼 제공

## 배포 규칙
- Vercel을 통해 손쉬운 배포
- SEO 최소화 적용 (title/meta/og)

## 확장성
- 추후 기능 확장 고려 (예: 설명문 자동 생성, 해시태그 추천)
- 컴포넌트 구조로 확장성 확보

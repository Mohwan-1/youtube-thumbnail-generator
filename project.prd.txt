# PRD – 썸네일 자동 생성기 (Thumbnail Auto Generator)

## 제품 개요
- 유튜버들이 영상 제목과 키워드만 입력하면 **AI(Google Gemini API)** 를 이용해 3~5개의 썸네일 문구/디자인을 자동 생성
- 마음에 드는 썸네일을 선택 후 **PNG/JPG 파일 다운로드**
- MVP 수준에서 빠른 배포와 **광고 수익화(AdSense)** 가 목적

## 목표
1. 최소 기능으로 빠르게 썸네일 자동 생성 서비스를 제공
2. 광고 수익을 통한 MVP 수익화 검증
3. 사용자 경험을 단순하고 직관적으로 유지

## 주요 기능
1. **입력/출력**
   - 입력: 영상 제목 + 키워드
   - 출력: 썸네일 3~5개 자동 생성 (텍스트 기반 디자인)
   - 다운로드: PNG/JPG 저장

2. **API 연동**
   - Google Gemini API 사용
   - 사용자가 직접 API KEY 입력
   - API KEY 입력 안내 버튼 제공 (Google Cloud Console로 연결)

3. **광고 배치**
   - 자동광고: 좌우 사이드바
   - 수동광고: 상단, 본문 중간, 하단

4. **디자인/브랜딩**
   - 다크모드 기본
   - 현대적인 미니멀 UI
   - 로고 + 파비콘 적용

5. **배포**
   - Vercel을 통해 배포
   - SEO 최소 설정 (title, meta description, OG 태그)

## 사용자 시나리오
1. 사용자가 사이트 접속
2. 영상 제목과 키워드를 입력
3. 썸네일 3~5개가 자동 생성
4. 마음에 드는 것을 선택 → PNG 다운로드
5. 광고는 자연스럽게 노출
